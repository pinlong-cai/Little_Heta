# heta mem-show

Inspect saved memory.

```bash
heta mem-show insights
```

Options:

```bash
heta mem-show insights --source pages/1-heta.md
heta mem-show insights --question "rate limits"
heta mem-show insights --limit 20
heta mem-show insights --full
```

This is mainly for inspecting KB insight memories created by `heta ask`.
